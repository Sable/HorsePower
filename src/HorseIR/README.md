
## Installation (MacOS)

Pre-requisites

- antlr4 (v.4.7, [download](http://www.antlr.org/download.html))
- homebrew

Install antlr4 runtime library

    brew install antlr4-cpp-runtime

Install testing suits

    brew install cxxtest
